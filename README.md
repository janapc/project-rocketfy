# Rocketfy
Project based on pipefy

### **Used Packages:**  *React*.

## Warning
 - **First problem, not possible drag card for list empty**
 - **Second problem, it is only possible to drag a card over another card and not above a list**


## Install
  - install package:
  ```
  yarn/npm i
  ```
  - Run application:
  ```
  yarn start/ npm start
  ```
